# RomanNumeralsPlus Resource Pack

This texture pack adds Roman numeral support for enchantment levels and potion potency levels (11-255) instead of numeric displays.

## Features

- **Enchantment Levels**: Replaces numeric enchantment level displays with proper Roman numerals
- **Potion Potency**: Replaces numeric potion potency levels with proper Roman numerals
- **Support for levels 11-255**
- Examples:
  - Instead of `Efficiency enchantment.level.255`, you'll see `Efficiency CCLV`
  - Instead of `Potion potency level 255`, you'll see `Potion potency CCLV`
  - Roman numerals use standard notation: C=100, L=50, X=10, V=5, I=1

## How It Works

The pack leverages existing Roman numeral characters already in Minecraft (XI, XVI, etc.) and extends support up to level 255 using standard Roman numeral notation for both enchantments and potions.

## Installation

1. Download or clone this repository
2. Place the resource pack in your `.minecraft/resourcepacks` folder
3. Enable the pack in your Minecraft resource pack settings

Enjoy cleaner enchantment and potion displays!
