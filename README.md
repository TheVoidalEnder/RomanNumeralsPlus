# RomanNumeralsPlus Resource Pack

This texture pack adds Roman numeral support for enchantment levels 11-255 instead of the ugly `Efficiency enchantment.level.255` display.

## Features

- Replaces numeric enchantment level displays with proper Roman numerals
- Support for levels 11-255
- Example: Instead of seeing `Efficiency enchantment.level.255`, you'll see `Efficiency CCLV` (Roman numerals: C=100, L=50, V=5, etc.)

## How It Works

The pack leverages existing Roman numeral characters already in Minecraft (XI, XVI, etc.) and extends support up to level 255 using standard Roman numeral notation.

## Installation

1. Download or clone this repository
2. Place the resource pack in your `.minecraft/resourcepacks` folder
3. Enable the pack in your Minecraft resource pack settings

Enjoy cleaner enchantment displays!
